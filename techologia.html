Lenguajes de programación

  • Python: sintaxis sencilla, ideal para data y scripts.
  • JavaScript: programación web frontend/backend.
  • Java: robusto para apps empresariales.
  • C#: usado en desarrollo Windows y videojuegos (Unity).

Frameworks y bibliotecas

  • React.js / Vue.js / Angular: para interfaces web dinámicas.
  • Node.js: JavaScript del lado del servidor.
  • Django / Flask: para web en Python.
  • .NET / Spring: para backend en C# y Java.

Otras herramientas esenciales

  • Git: control de versiones.
  • Docker: contenedores ligeros.
  • CI/CD: integración y despliegue continuo.
