¿Qué es programar?

Programar es escribir instrucciones que una computadora puede ejecutar. Es la base de todo el software...

Conceptos clave

  • Variables: espacios de memoria etiquetados.
  • Estructuras de control: if-else, loops, switch/case.
  • Funciones: bloques de código reutilizables.
  • Tipos de datos: números, strings, booleanos, arrays, objetos.
  • Algoritmos: secuencia de pasos para resolver problemas.
  • Estructuras de datos: listas, pilas, colas, árboles, grafos.

Ejemplo en JavaScript


// Calcula el factorial de un número
function factorial(n) {
  if (n <= 1) return 1;
  return n * factorial(n - 1);
}
console.log(factorial(5)); // 120
