¿Qué es HTML?

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje estándar que se utiliza para crear páginas web. HTML describe la estructura del contenido mediante etiquetas que indican los diferentes elementos, como títulos, párrafos, imágenes, enlaces y más.

¿Para qué se usa HTML?

  • Para estructurar el contenido de una página web.
  • Para insertar elementos como texto, imágenes, videos y formularios.
  • Para crear enlaces entre diferentes páginas o sitios web.
  • Para facilitar la accesibilidad y la organización del contenido.

Ejemplo de etiquetas HTML

Algunas etiquetas comunes de HTML incluyen:

  • <h1>...</h1>: para títulos principales
  • <p>...</p>: para párrafos
  • <a href="...">...</a>: para enlaces
  • <img src="...">: para imágenes
  • <ul><li>...</li></ul>: para listas

HTML + CSS = Sitios web modernos

HTML por sí solo crea la estructura, pero no es muy bonito visualmente. Para darle estilo a esa estructura, usamos CSS, que permite cambiar colores, tipografía, márgenes, animaciones y mucho más.
